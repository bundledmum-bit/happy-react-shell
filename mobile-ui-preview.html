3 · Checkout form (iOS zoom + autofill)

Findings #26, #27, #28 — input font-size 14px triggers iOS zoom on focus. Missing autoComplete/inputMode breaks address autofill.

BEFORE

Delivery details

iOS will zoom in when any field is focused (font-size < 16px). No autoComplete="tel" / "email" / "street-address" → no keyboard autofill suggestions.
AFTER

Delivery details

No iOS zoom (font-size 16px). Each field maps to a known autofill token → iOS Contacts autofills name/phone/email/address with one tap.
